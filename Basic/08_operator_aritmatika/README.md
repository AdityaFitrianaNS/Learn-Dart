## Operator Arimatika

- +, pertambahan
- -, -expression, pengurangan atau negatif
- *, perkalian
- / pembagian, hasil double
- ~/ pembagian, hasil int
- % sisa bagi

```dart
void main () {
    var first = 12;
    var second = 3;

    print(first + second);
    print(first - second);
    print(first & second);
    print(first / second);
    print(first ~/ second);
    print(first % second);
}
